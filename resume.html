  • Cofounder & Professor, The Code Liberation Foundation, 2013 - 2015

    Cofounded an organization that teaches women of all experience leves how to program games for free. Designed and developed open-source lesson plans, slides, and sample code in openFrameworks, C++, Processing, Unity, and GameMaker. Taught classes on basic game programming as well as more advanced subjects like Open Sound Control and Kinect.

  • Coder and Visual Designer, Melanie Crean, 2013 - 2014

    Designed and coded frontend for Melanie Crean's Memories of the Future, which was a smartphone app that led users on a walking tour of Hunts Point, South Bronx and used location tracking to play them spoken word poetry by residents at key points. Led visual design and UX.

  • Web Programming Intern, Kollabora, Summer 2013

    As part of selective hackNY Fellowship, developed collaborative filtering algorithm for new user recommendation feature.

  • Lead Level & AI Programmer, Golden Ruby Games, Summer 2013

    Designed and programmed levels and enemy AI in C# for Hermit Crab In Space, which later won the Sony GameJam and became an Indiecade nominee.

  • Game Design Intern, Playmatics, 2012 - 2013

    Designed levels and difficulty curves for compelling casual games.

  • Code Instructor, Parsons The New School For Design, Fall 2012

    Developed curriculum and taught intensive creative coding class in Processing to incoming MFA students.
